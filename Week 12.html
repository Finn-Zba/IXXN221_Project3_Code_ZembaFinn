Reflection

Wow this was a fun but harsh class

Personally, I really enjoyed this class. It was fun to learn and develop my skills as a designer by learning website-based design. It's a subject I've always been interested in, which makes this class something that has been important to me. This class has taught and increased my ability to provide both pleasing and effective web design as well as my goal to implement a potentially accessible element to it for those like me with neurodivergence. Being able to set the seeds to develop it further is something this class has enabled which is definitely the highlight for me personally. However, the pure amount of output required within such small hand-in periods led me to rush and leave some elements unsatisfactory to me, despite highly positive grades.
